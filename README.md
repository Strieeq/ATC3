# 🔐 Atomic TriFlow Cipher v3.0

Современный десктопный шифратор с симметричной криптографией, красивым интерфейсом и опциональной привязкой к железу.  
Разработан для безопасного хранения и передачи текстовых и бинарных данных.

## ✨ Возможности
- 🛡️ **Argon2id KDF** — вывод ключа из пароля, устойчивый к GPU/ASIC и side-channel атакам
- 🔒 **ChaCha20-Poly1305** — современное AEAD-шифрование (конфиденциальность + аутентификация)
- ✅ **BLAKE2b HMAC** — финальная подпись пакета, проверка целостности до расшифровки
- 💻 **Hardware Lock v2** — опциональная привязка к железу (CPU, мать, BIOS, MAC)
- 📁 **Работа с файлами и текстом** — шифрование/дешифрование в один клик
- 🎨 **PyQt6 GUI** — адаптивный интерфейс в молочно-лиловой палитре
- 🧩 **Модульная архитектура** — ядро, воркер, UI и стили разделены

## 📦 Установка
Требуется **Python 3.10+**.

```bash

# 2. Создай виртуальное окружение (рекомендуется)

source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. Установи зависимости
pip install -r requirements.txt
```

## 📖 Как пользоваться
| Вкладка | Описание |
|---|---|
| 🔒 **Шифрование** | Введи текст/выбери файл → задай ключ (≥16 символов) → нажми «Зашифровать». Результат в Base64. |
| 🔓 **Дешифрование** | Вставь Base64/выбери `.atc3` файл → введи ключ → «Расшифровать». |
| 🎲 **Ключи** | Генератор криптографически стойких паролей (32/64/128 символов). |
| ℹ️ **О шифре** | Описание алгоритмов, сравнение с MTProto, формат пакета. |

> ⚠️ **Важно про Hardware Lock:** Если галочка `Привязать к этому железу` включена, файл расшифруется **только на этом компьютере**. На другом устройстве будет ошибка `🔒 Файл привязан к другому железу или ключ неверный`.

## 🔐 Криптографическая схема
```
ATC3 | ver | flags | kdf_salt(32) | hw_salt(16) | nonce(12)
| ciphertext | poly1305_tag(16) | hmac_blake2b(32)
```
1. Генерируются случайные соли и nonce.
2. Из пароля через Argon2id выводится мастер-ключ.
3. Если включён HW Lock, к ключевому материалу добавляется отпечаток железа.
4. Финальный ключ = `BLAKE2b(master + hw)[:32]`.
5. Данные шифруются ChaCha20-Poly1305 (AEAD).
6. Весь пакет подписывается HMAC-BLAKE2b.

## ⚠️ Примечания по безопасности
- Проект использует **симметричную криптографию**. Ключ должен передаваться только по защищённому каналу.
- Алгоритмы взяты из проверенных библиотек (`pycryptodome`, `argon2-cffi`).
- `Hardware Lock` зависит от стабильности WMI/системных утилит. При замене железа или обновлении BIOS отпечаток может измениться.
- **Не предназначен для критически важных продакшен-систем без независимого криптографического аудита.** Отлично подходит для личных задач, архивов и обучения.
