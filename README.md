## 🔐 Криптографическая схема
```
ATC3 | ver | flags | kdf_salt(32) | hw_salt(16) | nonce(12)
| ciphertext | poly1305_tag(16) | hmac_blake2b(32)
```
1. Генерируются случайные соли и nonce.
2. Из пароля через Argon2id выводится мастер-ключ.
3. Если включён HW Lock, к ключевому материалу добавляется отпечаток железа.
4. Финальный ключ = `BLAKE2b(master + hw)[:32]`.
5. Данные шифруются ChaCha20-Poly1305 (AEAD).
6. Весь пакет подписывается HMAC-BLAKE2b.

## ⚠️ Примечания по безопасности
- Проект использует **симметричную криптографию**. Ключ должен передаваться только по защищённому каналу.
- Алгоритмы взяты из проверенных библиотек (`pycryptodome`, `argon2-cffi`).
- `Hardware Lock` зависит от стабильности WMI/системных утилит. При замене железа или обновлении BIOS отпечаток может измениться.
- **Не предназначен для критически важных продакшен-систем без независимого криптографического аудита.** Отлично подходит для личных задач, архивов и обучения.
